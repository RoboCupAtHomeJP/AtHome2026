# Open Challenge

## Description / チャレンジ概要

Teams demonstrate their latest research and innovative approaches.<br>
チームの最新の研究や革新的なアプローチを実演・デモンストレーションを行います．

---

## Local Variations / ローカル解説事項

The objective is set by each team. Japanese and English are allowed for the presentation.
目的は各チームが設定します．日本語または英語で発表します．

The score is not reflected in the main competition score. It is awarded individually.
このタスクのスコアはメイン競技のスコアには反映されず，個別に表彰されます．
