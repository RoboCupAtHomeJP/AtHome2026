# Poster Session

## Description / チャレンジ概要

This challenge follows **Section 4.3 (Poster Presentation Session)** of the [2026 RoboCup@Home Rulesbook](https://robocupathome.github.io/RuleBook/rulebook/master.pdf).<br>
このチャレンジは，[2026 RoboCup@Home Rulesbook](https://robocupathome.github.io/RuleBook/rulebook/master.pdf) の **Section 4.3 (Poster Presentation Session)** に準拠します．

---

## Local Variations / ローカル解説事項

- The presentation lasts for 3 minutes, with no Q&A session.
- 発表時間は3分，質疑応答は行われません．

- The presentation language is **English only**.
- 発表は**英語のみ**で行われます．

- The score is not reflected in the main competition score. It is awarded individually.
- このタスクのスコアはメイン競技のスコアには反映されず，個別に表彰されます．

- The jury is composed of one member from each team.
- 審査員は各チームから1名ずつ選出されます．
