# Poster Session

## Description / チャレンジ概要

This challenge follows **Section 4.3 (Poster Presentation Session)** of the [2026 RoboCup@Home Rulesbook](https://robocupathome.github.io/RuleBook/rulebook/master.pdf).<br>
このチャレンジは，[2026 RoboCup@Home Rulesbook](https://robocupathome.github.io/RuleBook/rulebook/master.pdf) の **Section 4.3 (Poster Presentation Session)** に準拠します．

---

## Local Variations / ローカル解説事項

セットアップデーに実施します。
各チーム研究内容等についてのポスター発表を行い、発表者以外のチームが以下の基準で評価します。
開始時間や場所等は後程アナウンスいたします。

【内容】
・発表時間は3分以内(質疑なし)
・スコアシートを元に各チーム互いに評価をします。ただし自チームの評価はできません。審査員は各チームから1名出してください。
・ポスターサイズはA0縦、各チームで印刷して持参してください。
・ポスターは必ず英語で作成してください。発表も英語で行って下さい。
・ポスターセッションのスコアは競技スコアに反映せず、個別で表彰されます。

【評価基準】
　**当日までに変更の可能性もあります。**1~5までの5段階評価です。
・新規性
・科学的貢献度
・RoboCup@Homeとの関連性
・ポスターや発表の質
