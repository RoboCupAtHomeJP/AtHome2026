# General Purpose Service Robot - GPSR

## Description / チャレンジ概要

This challenge follows **Section 5.3 (General Purpose Service Robot - GPSR)** of the [2026 RoboCup@Home Rulesbook](https://robocupathome.github.io/RuleBook/rulebook/master.pdf).<br>
このチャレンジは，[2026 RoboCup@Home Rulesbook](https://robocupathome.github.io/RuleBook/rulebook/master.pdf) の **Section 5.3 (General Purpose Service Robot - GPSR)** に準拠します．

---

## Local Variations / ローカル解説事項

GPSR
# 【重点】
ロボットは、多様な能力を必要とするコマンドを理解し、実行することが求められます。<br>
主な目標はオペレーターが発する3つのコマンドを実行することです。

# 【焦点】
タスク計画、物体/人物の検出と認識、物体の特徴認識、物体の操作

# 【セットアップ】
## 場所
原則タスクはアリーナ内で行われますが、コマンドによってはロボットが部屋から出る必要がある場合があります。アリーナは通常構成になっています。<br>
 ロボットはアリーナの外から開始し、初めに指示ポイントに向かって移動する必要があります。ロボットはすべてのコマンドを完了した後、指示ポイントに戻ります。

## プロのオペレーター
審判員がロボットに標準的なコマンドを発します。3回程度試行したとしてもロボットがコマンドを理解できない場合、チームはオペレーターを変更することができます。

# 【手順】
##  指示ポイント
テストの少なくとも2時間前に、審判員は指示ポイントの場所を発表します。
## コマンド実行
ロボットはコマンドを連続して発行するか、1つずつ発行するかを決定しオペレーターに伝える必要があります。コマンドが1つずつ発行される場合、ロボットは各タスクを完了するたびにオペレーターの元に戻らなければなりません。
## 指示ポイントへの帰還
ロボットはオペレーターから与えられたすべてのコマンドを完了した後、指示ポイントに戻ります。
## タイマーの一時停止
ロボットが指示ポイントに到達後審判はタイマーを一時停止し、次のコマンドのためにアリーナをリセットします。審判が次のコマンドの開始を合図すると、タイマーは再開します。

【備考】
コマンド等に関しては現在調整中です。<br>

# 【減点要素】
カスタムオペレーターの使用。<br>
音声認識のバイパス。<br>
コマンド実行のための人間の支援。<br>
人間のアシスタントにタスク全体を実行するよう指示すること。<br>

審査員はオペレーターにコマンドを提供する必要があります。<br>
このテストの最大時間は7分です。

# 【スコアシート】
音声コマンドを理解する 3×80<br>
計画が生成されたことを示す 3×100<br>
最初のコマンドを解決する 250<br>
2番目のコマンドを解決する 250<br>
3番目のコマンドを解決する 250<br>
コマンドの一部のみを解決した場合、他のテストでの同様のスコアに応じて、パーセンテージによるペナルティが適用されます。<br>
連続ボーナス 200<br>
合計：1250<br>

【減点要素】
カスタムオペレーターを使用する 3×–20<br>
言い換えを要求する 6×–30<br>
音声認識の回避 3回 – 50<br>
不参加 – 500<br>
※もともとGPSRに参加予定のないチームはこれに該当しません。<br>
代替開始信号の使用– 100<br>
→開始ボタン1回押す、緊急停止ボタンの解除以外の行動は全て代替開始信号とみなします。これら以外の行為はセットアップの時間に行ってください。<br>
